# Завдання

## Рівень 1

6. Створіть простий відеохостинг на Node.js з використанням Express.  
Реалізуйте фронтенд на React для відображення відео.

## Рівень 2

6. Додайте можливість завантаження та перегляду відео.  
Реалізуйте систему коментування відео.

## Рівень 3

6. Реалізуйте систему користувачів та управління їхніми правами.  
Додайте функціонал "Поділитися відео", щоб користувачі могли надсилати відео іншим користувачам.

## Рівень 4

6. Оптимізуйте систему для високих навантажень.  
Реалізуйте систему підписки на канали та повідомлень про нові відео.
